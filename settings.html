Please read through these structural execution behaviors carefully before spinning up your node.

📊 Tactical Trading Matrix

  • ▪ Execution Interval: The engine performs its data analysis once per trading day. It prioritizes daily trends over noisy intraday moves.
  • ▪ Position Multi-Max: The bot can manage positions across multiple pairs simultaneously, running a maximum of 1 high-probability trade per pair each trading day.
  • ▪ Strict Risk Management: Every single open position dynamically applies a strict 2% capital risk floor to preserve your balance.
  • ▪ No Take-Profit Targets: This engine is designed without arbitrary profit caps. It rides massive trends indefinitely, automatically locking in profits using dynamic trailing stops as the asset runs.

🔑 How to Find Your Bybit API Credentials

  1. Log into your Bybit Account, hover over your profile icon in the top-right corner, and select API.
  2. Click Create New Key → Select System-Generated API Keys.
  3. Set API Key Usage to Connect to Third-Party Applications.
  4. Set the access permissions to Read-Write.
  5. Under the Unified Trading Account (UTA) column, check the boxes for Order and Positions. ⚠️ LEAVE 'WITHDRAWAL' UNCHECKED. The bot never needs permission to move your funds.
  6. Pass your 2FA verification security wall, then copy your API Key and API Secret over to the setup vault form on the right immediately.
